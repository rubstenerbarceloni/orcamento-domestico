Orçamento Doméstico Local - PWA para iPhone

Arquivos:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png

Como usar no iPhone:
1. Hospede esta pasta em um servidor HTTPS simples.
2. Abra o index.html no Safari.
3. Toque em Compartilhar > Adicionar à Tela de Início.
4. O app passará a abrir em modo parecido com nativo.

Observação importante:
No iPhone, o modo PWA instalável precisa ser aberto via Safari a partir de um endereço web seguro (HTTPS).
Abrir o arquivo solto pelo app Arquivos pode não ativar todo o comportamento de PWA, especialmente o service worker.

Opções simples para publicar:
- GitHub Pages
- Netlify
- Vercel
- qualquer hospedagem estática com HTTPS

Os dados ficam salvos localmente no navegador do aparelho.
