Manual rápido

Dashboard BI

  • Use a aba Dashboard BI para ver gráficos de saldo, categorias, comparativo mensal e projeções.
  • Os gráficos seguem o filtro do período.

Previsões

  • No cadastro, selecione Lançamento futuro / previsão.
  • Esses eventos entram no fluxo projetado e nos insights.

Sincronização

  • O app guarda localmente e sincroniza com Supabase após login.
  • Use Sincronizar agora quando quiser forçar atualização.

Recorrências

  • Use parcelas para compras parceladas.
  • Use permanente para salários, aluguéis ou mensalidades.
