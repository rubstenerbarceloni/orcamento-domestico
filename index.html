Manual integrado

1. Configuração inicial

  • Preencha o arquivo supabase-config.js com a URL e a anon key do seu projeto.
  • Execute o conteúdo de supabase_schema.sql no SQL Editor do Supabase.
  • Publique a pasta no GitHub Pages.

2. Conta

  • Use Criar conta para registrar usuário.
  • Use Entrar para acessar seus dados na nuvem.
  • Use Recuperar senha para redefinir a senha por e-mail.

3. Lançamentos

  • Cadastre receitas ou despesas com descrição, categoria, valor e data.
  • Ative recorrência para parcelas ou lançamentos permanentes.
  • Edite ou exclua lançamentos a qualquer momento.

4. Fluxo e relatórios

  • Use o filtro por período para recalcular o painel, o gráfico e os relatórios.
  • O fluxo de caixa mostra o saldo acumulado após cada evento.

5. Sincronização

  • O app guarda dados localmente e sincroniza com a nuvem após o login.
  • Use Sincronizar agora para enviar e baixar dados.
  • Quando estiver sem internet, continue usando normalmente; a sincronização pode ser feita depois.

6. Backup

  • Exporte backup em JSON para guardar uma cópia externa.
  • Importe o JSON para restaurar os dados.
  • O manual completo também está disponível em PDF.
